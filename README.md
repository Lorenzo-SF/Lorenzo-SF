### 🔧 Open Source — Elixir Ecosystem

A family of libraries built to solve real problems I encountered working with Elixir every day.

| Project | What it does |
|---------|-------------|
| [**Arrea**](https://github.com/Lorenzo-SF/arrea) | OTP async process orchestrator with circuit breaker, retry policies and telemetry |
| [**Candil**](https://github.com/Lorenzo-SF/candil) | LLM inference for Elixir — local models via llama.cpp and remote via OpenAI, Anthropic, Ollama |
| [**Alaja**](https://github.com/Lorenzo-SF/alaja) | Declarative CLI framework and terminal rendering kit with DSL macros |
| [**Delfos**](https://github.com/Lorenzo-SF/delfos) | MCP server that indexes your codebase and exposes it to AI assistants (Claude, Cursor, Zed) |
| [**Flotilla**](https://github.com/Lorenzo-SF/flotilla) | Declarative wrapper for Phoenix LiveView following The Elm Architecture |
| [**Válvula**](https://github.com/Lorenzo-SF/valvula) | Zero-dependency token bucket rate limiter built on OTP/ETS |
| [**Apero**](https://github.com/Lorenzo-SF/apero) | Utility library — file ops, Git, Docker, crypto, HTTP, retry, cache |
| [**Pote**](https://github.com/Lorenzo-SF/pote) | Color manipulation library — parsing, conversion, palettes, xterm256 |
| [**Botica**](https://github.com/Lorenzo-SF/botica) | Health checks and environment diagnostics with parallel execution |
| [**Batamanta**](https://github.com/Lorenzo-SF/batamanta) | Package Elixir apps as self-contained executables with embedded ERTS |

All packages available on [hex.pm](https://hex.pm/users/lorenzo-sf).
